## Próximos passos (coleta física das imagens)

Essas URLs são **pontos de entrada curatoriais** — não as imagens em si. Para poder usar como *style reference* no Gemini, um operador humano (ou outra sessão Claude com navegador headless) precisa:

1. Abrir cada URL listada
2. Salvar 1-3 imagens representativas em `docs/refs/xilogravura/NN_autor_obra_curto.jpg` (manter numeração da tabela)
3. Registrar em `docs/VISUAL_LICENSES.md` a licença de cada imagem baixada (dominio público, CC-BY, uso editorial, etc.)
4. Flagear imagens que exigem permissão direta (obras recentes de Samico/Borges — alguns podem estar protegidos pela família/galeria mesmo após 2024)

Não subir ao repositório imagens que não tenham licença compatível (CC0, CC-BY com atribuição, ou domínio público). Para essas, guardar link e descrição textual, usar só como *inspiração de prompt* (não como *style ref* literal).

## Paletas de inspiração extraídas

As referências acima sustentam a paleta definida em `docs/palette.md`. Principais observações cromáticas:

- Xilogravura tradicional: papel cru + tinta única (preto) — base mais monocromática do que a que o GDD propõe
- Cordéis coloridos pós-1970 (J. Borges maduro): adicionam vermelho-tijolo, ocre, verde-folha, índigo — coincide com a paleta já em `ART_BIBLE.md` §3
- Samico: paleta mais restrita ainda, preto + ocre + índigo. Usar para bosses (peso icônico)
- Carnaval/frevo: quebra da paleta — rosa-fosco + amarelos vibrantes, só em picos (power-ups, hit feedback)
