# docs/refs/xilogravura/

Armazenar aqui as imagens de referência curadas em `../REFS.md`.

## Convenção de nome

`NN_autor_obra.ext` — onde `NN` bate com o número na tabela de `REFS.md`.

Exemplos:
- `01_jborges_vida_na_floresta.jpg`
- `09_samico_a_arvore_da_vida.png`
- `20_homem_meia_noite_2026.jpg`

## Licenças

Toda imagem salva aqui **precisa** ter entrada correspondente em `docs/VISUAL_LICENSES.md` com fonte, autor, licença e link. Imagens sem licença clara não entram — ficam só como link em `REFS.md` para consulta.

## Uso

Essas imagens alimentam o pipeline de IA como *style reference* (Gemini 2.5 Flash Image aceita imagens multimodais no prompt). Não são usadas diretamente no jogo.
